# borntofrappe

> Project restarted on January 10th 2023. At 10:02 PM. With SvelteKit 1.0.

## routes

- `/`: landing page

- `/[slug]`: blog post

- `/blog`: blog API

- `/feed.xml`: Atom feed

- `/color-tiles`, `/minesweeper`, `/rainbow-sixteen`: showcase

- `/mignon`: mascot

- `/picross-svg`: solved nonogram puzzles

- `/tangram`: tangram shapes

- `/today`: current day
