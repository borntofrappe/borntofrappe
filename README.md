# borntofrappe

> svelte, sapper, netlify

## What's this

Personal website. With a landing page and a blog.

## Up and Running

```code
npx degit "sveltejs/sapper-template#rollup" borntofrappe
cd borntofrappe
npm install
npm run dev
```
